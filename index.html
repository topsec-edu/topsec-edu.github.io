为什么要学习GitHub

GitHub 对于技术开发人员来说很有帮助，上面有很多优秀的开源项目。

安装易语言

下载易语言安装包

https://pan.baidu.com/s/1FvmgVdjbYQqc_TCupcI0Ag

1、双击：易语言5.71安装包.exe

要记好安装路径，比如：C:\Program File\e\e.exe

这是易语言的安装包，安装好后还要破解。

2、破解：把文件夹（将此文件夹下的文件复制到易语言文件夹目录）里的内容复制到安装路径下，选择替换已有文件。

这样你的易语言就破解完了。
